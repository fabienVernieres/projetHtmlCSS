Code fichier : index.html

  1. Balises de base html, head et body
  2. Balises title pour le titre de la page
  3. Meta charset pour déclarer l'encodage du document en UTF8
  4. Balise meta viewport pour contrôler la mise en page sur mobile
  5. Balise link pour lier le fichier CSS
  6. Balises structurantes : header, nav, section, aside, footer
  7. Commentaires et repères
  8. Header : logo et menu
  9. Section : contenu principal et contenus secondaires
  10. Footer : liens, informations et divers
  11. Attributs id et class sur les balises
  12. Tableaux pour les contenus secondaires
  13. Mise en relief avec la balise strong du contenu
  14. Validation du code w3c

Code fichier : style.css

  1. Définition de la police de caractère avec @font-face
  2. box-sizing: border-box pour éviter que le remplissage et les bordures n'influent sur la taille des différents blocs
  3. Choix des couleurs, taille de la police, apparence des liens
  4. Utilisation de la propriété flex pour le header, la section principale et le footer
  5. Taille des blocs de contenu
  6. Effet visuel zoom pour les 3 asides secondaires
  7. Mise en forme des tableaux
  8. Réduction des marges, direction du menu et gestion de la bannière selon largeur d'écran
  9. Validation du code w3c
